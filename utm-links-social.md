# Ссылки на сайт для соцсетей (с UTM-метками)

Готовые ссылки на mortgageservicepro.com для профилей соцсетей. Каждая ссылка помечена
UTM-метками — благодаря им мы видим в аналитике, из какой соцсети пришёл посетитель и какие
заявки она принесла. Для посетителя ссылка работает как обычная.

**Просьба: ставить ссылки именно в таком виде, ничего в них не меняя.** Скопировать блок
целиком — вставить в указанное поле.

---

## Facebook

```
https://mortgageservicepro.com/?utm_source=facebook&utm_medium=social&utm_campaign=bio
```

Бизнес-страница → «Редактировать данные» → «Контактная и основная информация» → поле
**«Веб-сайт»**. Если личный профиль: «Редактировать профиль» → «Ссылки» → добавить ссылку.

- [ ] Поставлено

## LinkedIn


## YouTube

```
https://mortgageservicepro.com/?utm_source=youtube&utm_medium=social&utm_campaign=bio
```

YouTube Studio → «Настройка канала» → вкладка «Основные сведения» → раздел **«Ссылки»** →
добавить (появится кнопкой в шапке канала). Плюс рекомендация: вставлять эту же ссылку
**в описание каждого видео и Shorts** — из описаний переходов больше, чем из шапки.

## WhatsApp — личный (бизнес-профиль)

```
https://mortgageservicepro.com/?utm_source=whatsapp&utm_medium=social&utm_campaign=bio&utm_content=profile
```
