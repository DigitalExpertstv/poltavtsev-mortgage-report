## Куда и как поставить

**Facebook (бизнес-страница).** Страница → «Редактировать данные» → «Контактная и основная
информация» → поле **«Веб-сайт»**. Если речь о личном профиле: «Редактировать профиль» →
«Ссылки» → добавить ссылку.

**Instagram.** «Редактировать профиль» → **«Ссылка»** (или «Ссылки» → «Добавить внешнюю
ссылку», если уже стоит другая). Это единственное кликабельное место в Instagram — важно,
чтобы ссылка стояла именно там.

**LinkedIn.** Свой профиль → «Контактная информация» → карандаш → «Веб-сайт» → добавить.
Дополнительно можно вынести ссылку карточкой в блок «Избранное» (Featured) — она будет видна
сразу под шапкой профиля.

**YouTube.** YouTube Studio → «Настройка канала» → вкладка «Основные сведения» → раздел
**«Ссылки»** → добавить (появится кнопкой в шапке канала). Плюс рекомендация: вставлять эту же
ссылку **в описание каждого видео и Shorts** — из описаний приходит больше переходов, чем из
шапки.

**TikTok.** «Редактировать профиль» → поле **«Сайт»**. Важно: кликабельная ссылка в TikTok
доступна только бизнес-аккаунтам. Переключиться бесплатно: Настройки → «Аккаунт» →
«Переключиться на бизнес-аккаунт».

**Telegram — канал.** Открыть канал → «Изменить» → **«Описание»** → вставить ссылку текстом
(в описании канала ссылки кликабельны). Дополнительно можно закрепить пост со ссылкой.

**Telegram — личный профиль.** Настройки → «Изменить профиль» → поле **«О себе»** → вставить
ссылку (в актуальных версиях Telegram она кликабельна).

**WhatsApp — личный.** У обычного аккаунта WhatsApp кликабельного поля для сайта нет.
Правильный способ — бесплатное приложение **WhatsApp Business** (работает с тем же номером):
Настройки → «Инструменты для бизнеса» → «Профиль» → поле **«Сайт»**. Эту ссылку увидит и
сможет открыть каждый, кто зайдёт в профиль.

## Как проверить, что работает

Открыть любую из ссылок самому — и в течение минуты визит появится в Google Analytics
(«Отчёты → В реальном времени») с указанием соцсети-источника. Заявки, пришедшие по этим
ссылкам, в уведомлениях Telegram будут приходить с пометкой источника (например
`source=instagram`).
